Installing on the Mac:



Get Swig:

	Get swig from:
		http://sourceforge.net/project/downloading.php?groupname=swig&filename=swig-1.3.31.tar.gz&use_mirror=superb-east
		
	or:
		http://www.swig.org/download.html
		
	cd swig-1.3.31
	./configure
	make
	sudo make install



Get Libraries:

	I also used MacPorts to install an up-to-date version of glib2 (2.12.12):
	
	http://www.macports.org/
	
	http://geeklair.net/new_macports_guide/
	
	sudo port install glib2



In Bash shell:

	export PATH=/opt/local/bin:/usr/local/bin:$PATH

	You may also need:
	
	export MACOSX_DEPLOYMENT_TARGET="10.4"

	Use the Mac installer
